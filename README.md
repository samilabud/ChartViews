# Chart View

Project created to show how the chart.js and lightweight-charts libraries work in React, specifically using the Next.js framework.

## Demo

https://chartviews.onrender.com/

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.
